### Paper Rock Scissors Game - Basic version

#### Languages employed:
- JavaScript / HTML
- Tested with Mocha/Chai
- Deployed on Netify

#### Deployed site:


As a player I would like to play the RPS Game.
The win conditions are as follows:
- Rock > Scissors
- Paper > Rock
- Scissors > Paper

#### INSTRUCTIONS:
- Clone from Github
- Run: npm test specs / features
- Run: npm run server
- Navigate to http://localhost:3000
